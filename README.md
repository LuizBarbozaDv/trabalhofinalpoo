### Descrição do Projeto
Sistema de Agenda de Contatos desenvolvido em **Java** com interface gráfica **Swing**.  
O objetivo é gerenciar contatos com cadastro, edição, exclusão, busca, filtros avançados (favoritos, categoria, aniversariantes do mês)

### Principais Funcionalidades
Cadastrar Contato:
 O sistema deve permitir o cadastro de novos contatos com nome, telefone, e-mail e categoria.
Listar Contatos:
 O sistema deve exibir todos os contatos cadastrados em ordem alfabética, categoria ou favoritos.
Buscar Contato:
 O sistema deve permitir a pesquisa de contatos utilizando o nome completo ou parcial.
Atualizar Contato:
 O sistema deve permitir editar as informações de um contato já existente.
Excluir Contato:
 O sistema deve permitir remover um contato do sistema.
Detalhar Contato:
 O sistema deve exibir informações completas do contato selecionado.
Validar Campos Obrigatórios:
 O sistema deve validar se nome e telefone foram informados antes de salvar.
Filtrar Aniversariantes:
 O sistema deve permitir filtrar contatos que fazem aniversário no mês atual.
Favoritar Contatos:
 O sistema deve permitir que o usuário adicione contatos a uma lista de favoritos.
Criar Categorias:
 O sistema deve permitir que o usuário crie categorias personalizadas para adicionar aos seus contatos.
