### Descrição do Projeto
Sistema de Agenda de Contatos desenvolvido em **Java** com interface gráfica **Swing**.  
O objetivo é gerenciar contatos com cadastro, edição, exclusão, busca, filtros avançados (favoritos, categoria, aniversariantes do mês)

### Principais Funcionalidades
- Cadastro e edição de contatos (com validação)
- Marcação de favoritos com um clique
- Filtro por nome, categoria e favoritos
- Filtro de aniversariantes do mês atual
- Campo Categoria

### Tecnologias Utilizadas
- **Linguagem:** Java (versão 24)
- **Biblioteca gráfica:** Swing
